# Sabikonaeh

A web application for text paraphrasing and summarization.

**Note:** The AI, powered by OpenAI, may have slow responses due to high request volume. Please be patient.

## Tech Stack

### Frontend
- React: JavaScript library for building user interfaces
- TypeScript: Typed superset of JavaScript for better code quality
- Vite: Fast build tool and development server
- DaisyUI: Component library for Tailwind CSS
- Lucide React: Icon library for React
- ReactBits: Library for React components and animations

### Backend
- Node.js: JavaScript runtime for server-side development
- Express.js: Web framework for Node.js

### Deployment
- Vercel: Platform for deploying frontend and backend applications
